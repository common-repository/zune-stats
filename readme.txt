== Description ==

Displays your Zune Stats on your sidebar with a choice of displaying either 
of the following:

* Recently Played Tracks
* Most Played Artists
* Favourite Tracks

View a demo [here](http://www.zmastaa.com/zunestats "Demo")


== Installation ==


1. Upload the folder `zunestats` to your `/wp-content/plugins/` directory

2. Activate the plugin

3. To configure the settings go to your widget selections panel `wp-admin/widgets.php` 
	and select ZuneStats
